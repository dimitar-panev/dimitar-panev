Hey! I'm a Network and Systems Administrator and IT student currently deep-diving into the world of infrastructure. While I have a background in multiple programming languages (C#, Python, Web Development, JS), my daily activities involve network topologies, configuring devices, and ensuring server stability.

I’m particularly interested in the practical side of hardware but I don't mind investing time in software and cloud architecture as well :D — I spend a lot of time working with MikroTik, EnGenius, Grandstream, and Milesight gear. Whether it's troubleshooting a sudden physical hardware power failure, analyzing missing core kernel error logs, or configuring a new subnet, I’m all about logical deduction and efficient systems. Currently, I am exploring the world of DevOps, so you will find me working on various projects that include containerization, self-hosting, and automation.
